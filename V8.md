Build v8

### Install depot_tools

```shell
git clone https://chromium.googlesource.com/chromium/tools/depot_tools.git
```

Add `PATH=/path/to/depot_tools:$PATH` to ~/.profile

### build v8

```shell
mkdir v8
cd v8
fetch v8
gclient sync
cd v8
./build/install-build-deps.sh #linux only
tools/dev/gm.py x64.debug --gn-args='v8_use_external_startup_data=false'

./out/x64.debug/d8  --code-comments  --print-code  index.js
```

For release build
```shell
tools/dev/gm.py x64.release --gn-args='v8_use_external_startup_data=false'
```
